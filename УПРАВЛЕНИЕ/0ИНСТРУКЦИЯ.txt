УСТАНОВКА:

1. Установите Python с помощью "1. Установка Python"
2. Убедитесь в коректности установки запустив "2. Проверка версии Python"
3. (Правая кнопка мыши -> Запуск от имени Администратора) Установите библиотеки с помощью "3. Установка библиотек BACKEND"

ЗАПУСК WINDOWS:

4. Узнайте свой IP-адрес внутренней сети с помощью "4. Узнать свой IP"

192.168.x.x - чаще WiFi --- как раз он нам и нужен

(в редких случаях 172.x.x.x 10.x.x.x)

Если их несколько - то запустите файл "ipconfig" и найдите именно тот адаптер,
с которого производится подключение (прим. TP-Link, Intel, Адаптер беспроводной сети Беспроводная сеть)

Если возникли трудности, то в Windows 10 можно проверить с помощью нажатия на значок WiFi, нажать на Имя_подключенного_wifi, нажать Свойства, в открывшемся окне пролистать вниз и посмотреть на IPv4 адрес

5. Запустите Backend с помощью "Запуск BACKEND" введя полученный IP


Разворачивание на сервере:

Сценарий:

1) Установите nginx
2) Установите gunicorn
3) Перекиньте файлы с папкой в /var/www/__FILENAME__
4) Настройте gunicorn.socket и gunicorn.service
5) Настройте nginx
6) Проверьте работу
